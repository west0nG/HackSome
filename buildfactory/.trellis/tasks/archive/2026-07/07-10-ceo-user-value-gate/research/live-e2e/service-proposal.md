# 候选公司 Objective：人工 MCP 配置审计服务

公司拟提供一次性人工服务，而不是继续开发软件。具体用户是一家 10–40 人 SaaS 团队的工程负责人：团队刚允许多个编码 agent 和 MCP server 进入项目，准备合并新的共享配置，但没有专职安全工程师。触发是首次合并共享 MCP 配置或一次较大配置变更。

服务交付是：客户提供脱敏配置和当前审批要求，公司在一个工作日内人工检查本地命令、包与镜像固定、token 范围、文件权限、远程 endpoint 和 registry 边界，交付一份带优先级的整改清单，并进行一次 30 分钟复核。无需构建新软件；现有 CLI 只能作为审计辅助工具。用户当前的替代是工程负责人自己花约 30–60 分钟逐项查看、请同事 review、使用现有平台控制，或者不做。

候选收费设想为首次审计 300 美元。第一批用户拟从已经公开讨论 MCP 安全的工程团队和现有 GitHub 反馈入口中触达。承重证据只在下列文件中，评审者应亲自打开：

`/Users/weston/dev/BuildFactory/.trellis/tasks/07-10-ceo-user-value-gate/research/live-e2e/service-evidence.md`

关键假设是目标负责人会把人工审计委托给外部服务，而不是继续自己查看。最小改变判断的结果是一支符合画像的外部团队提供脱敏配置并完成一次真实试审或付费委托；若没有样本、试审或委托行为，则调整触达/用户或终止。
