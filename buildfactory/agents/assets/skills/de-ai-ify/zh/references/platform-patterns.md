## H. 平台场景与文体

### 37. 自媒体 / 社交平台文案套路

AI 生成的**公众号、小红书、短视频口播、朋友圈**各有高辨识度套路。这是**消费端文本专属的 AI 腔**，专门覆盖自媒体语体。

> 2021 年那种老 AI 自媒体腔(震惊体 / 绝绝子 / 姐妹们冲)在 2023-2024 一度被**伪疗愈 / 伪深度**取代,但 2025-2026 观察到**AI 小红书生成又大量复用老模板** —— `🆘 救命 / 家人们谁懂啊 / 姐妹们快冲 / 不好吃回来打我` 和新伪疗愈腔**并存叠加**出现。判断时**新老套路都要扫**,两套都用密度 + 结构触发。本节先按平台列出老套路(现在仍大量出现),再列当下主流的伪深度 / 伪疗愈 / 伪钩子。跟本文件下文流行语表里"出现即高警觉"的口径统一。

**A. 公众号：**

**老套路（已减少，但偶现）：**
- `震惊!99% 的人都不知道的...`
- `原来 X 竟然可以这样`
- `紧急提醒:再不 X 就晚了`

**新 AI 腔：伪深度咨询腔**（2024-2025 主流）
- **咨询黑话堆砌：** 逻辑闭环、颗粒度、心智、叙事、底层（逻辑）、生态位、长期主义、信息茧房、认知突围、复利、飞轮、杠杆、势能、第一性原理、范式转移
- **结构固定：** 抛出普遍焦虑（职场 / 婚恋 / 搞钱）→ 引入 2-3 个术语（不解释原意，直接挪用）→ `不是 X，而是 Y` / `拉开差距的其实是` / `底层逻辑是` 做伪洞察 → 结尾升华到"长期主义 / 自我迭代 / 认知升级"
- **诊断关键：** 抽掉术语后**没有任何具体事实或私人经验** —— 只剩正确的废话。真人写商业洞察会带具体公司、具体数字、具体事件；AI 写全是术语对术语的空转。
- **判断法：** 随手挑一段，替换术语成同义空泛词（"逻辑闭环" → "想明白"），看看有没有掉信息。没掉 —— 就是伪深度。

**格言公式（humanizer #32 中文版）：** 把论点包装成可转发金句 —— `XX 是成年人的 YY` / `XX 是当代人的 ZZ` / `XX 本质上是一场 YY` / `XX 是另一种 YY`。显深刻、实空洞。
- **判定：** 密度敏感 —— 短段反复套「X 是 Y 的 Z」格言模板；换主语仍成立；落点是抽象拔高。跟 #48（否定对举）、#22（排比段）联动。
- **⚠ 不触发（同 #48 落点法）：** 单次、且 Y/Z 是具体去光环的真相，保留。
  - ❌ AI 腔：`断舍离是成年人的精神修行` —— Y 抽象拔高
  - ✅ 好句（保留）：`降薪是成年人的体面` —— Y 去光环的具体真相

**B. 小红书：**

**老套路（2025-2026 仍大量复用,不是"绝迹"）：**
- `姐妹们快冲!这个真的绝了💖` / `不好吃回来打我` / `闭眼入就对了`
- `家人们谁懂啊!` + 密集 emoji `✨🎀💖🌸👍🔥`
- 刻意惊呼:`救命! / 慌了! / 天哪!` / 开头 `🆘` emoji 替代文字版 `救命`
- **夸张量化吹牛**:`显瘦 10 斤不是梦 / 准时下班不是梦 / 一周重回好皮肤 / 人均 50r 吃到撑`
- **万能赞美词堆叠**:`宝藏 / 神器 / 天花板 / 绝了 / 绝绝子 / 绝绝 / 好吃到想原地开店 / 气场两米八 / 小心机`—— 短段 <300 字 ≥3 处即触发,脱离具体内容、换任何话题都能套
- **身份标签 + 反向 CTA 组合**:`姐妹们快冲` / `不好吃回来打我` / `打工人必看` / `租房党必看` / `学生党必看` / `敏感肌姐妹` / `宝妈必看` —— 身份标签 + `快冲 / 闭眼入 / 必看 / 必入 / 码住` 组合高频,密度 ≥2 组即触发
- **info-block emoji 微模板**(槽位化四字小标题 + emoji 前缀):
  - 探店:`✅ 必点招牌 / 📍 地址 / ⏰ 营业时间 / 💰 人均`
  - 好物:`✅ 好物清单 / 💰 价格 / 📦 使用场景`
  - 护肤:`🌟 晨间护肤 / 🌙 晚间护肤 / 💡 Tips / ❗ 避雷`
  - 教程:`✅ 技巧 1 / ✅ 技巧 2 / ✅ 技巧 3`
  - 跟 #18 过度数字化列点(1️⃣2️⃣3️⃣)分工:#18 是**编号**模板,本条是**命名 + emoji 前缀**模板。两条叠加即高警觉。

**区分真·小红书俚语(保留)vs AI 堆叠(改):**

- 保留:`闭眼入 / ins 风 / 码住 / 种草 / 精致 / 小心机`(单独出现,真人高频,保留;跟 `city walk / emo / PUA` 同类)
- 改:`姐妹们快冲!不好吃回来打我!`(真人不会这么连堆,AI 模板化)
- 改:`租房党必看 + 闭眼入 + 幸福感爆棚`(身份标签 + 俚语机械叠加)
- 改:`宝藏 × N / 神器 × N / 天花板 × N`(密度过高,万能赞美词脱离内容)

**新 AI 腔:伪疗愈 / 伪搞钱 / 伪清醒**(2024-2025 主流,与上面老套路并存)
- **开头三件套：** 痛点 + 身份标签 + 时间锚点
  - `25 岁裸辞第 30 天，我终于明白...`
  - `建议所有内核不稳的女生反复观看`
  - `30 岁之后我才懂，真正拉开差距的是...`
- **灵性 / 心理学黑话稀释版：**
  - 允许一切发生、把自己重养一遍、能量场、课题分离、内核稳定、高能量人格
  - 戒掉讨好型人格、MBTI 平替（`i 人 / e 人 / 淡人 / 浓人 / 确诊为 xx`）
- **搞钱黑话：** 复利思维、认知杠杆、长期主义、睡后收入、反内卷
- **排版特征：** 每段开头一个 emoji，三段式（痛点 → 方法 → 升华），结尾必带 `建议收藏 / 慢慢看 / 评论区告诉我你的阶段`
- **诊断关键：** **正确的废话 + 排比句 + 情绪平滑曲线**，没有具体崩溃瞬间、具体账单、具体失败。真人写"把自己重养一遍"会带具体的"我今天花 80 块吃了顿饺子"；AI 写全是抽象的"好好爱自己"。

**C. 朋友圈装 X 文案（凡尔赛式）：**
- `今天的阳光真好☀️,随手拍了一张📷,没想到竟然火了🔥`
- 刻意"无心" + 滤镜 + 留白
- `今天开了个会，老板把我从 xx 提到 xx，心情复杂` 这种

**D. 短视频口播（2025 算法驯化版）：**

**老套路（基本死透）：**
- `今天给大家分享一个超级实用的小技巧`
- `不看后悔!看完赶紧转发给身边的朋友`

**新 AI 腔：算法钩子体**
- **玄学 / 大数据钩子：**
  - `刷到这条视频，说明你马上要...`
  - `大数据不会乱推，留下"接"字...`
  - `能刷到这条的人都不简单`
- **恐吓 / 反常识钩子：**
  - `千万别再 X 了! 90% 的人都在犯这个错`
  - `我劝你立刻停止 X`
  - `这三件事，你现在知道还不晚`
- **电影解说腔（AI 重灾区，永不停歇）：**
  - `注意看，这个男人叫小帅 / 小美...`
  - `这个看似不起眼的动作，背后藏着...`
- **节奏特征（AI 配音可辨）：**
  - 句长被严格控制在 **3-5 秒一个断句**（方便 AI 配音换气）
  - 每 15 秒必有一个**情绪转折点**
  - 结尾 CTA 机械化：`点赞收藏，不然刷着刷着就找不到了 / 关注我，带你了解更多`

**E. AI 客服"您好呢亲":**
- `您好呢亲~很高兴为您服务哦!`
- `亲亲这边建议您...`
- `亲,这个问题小助手已经记下啦~`

**改写方向：**
- **伪深度咨询腔** → 抽掉术语后能不能保留信息？不能就整段重写，换成具体公司 / 数字 / 事件
- **伪疗愈 / 伪搞钱** → 把抽象升华换成具体细节。"好好爱自己" → "今天给自己买了一包贵的烟"
- **算法钩子 / 电影解说腔** → 删钩子，直接给内容；本来就没内容，建议别发
- 删密集 emoji、机械 CTA、刻意惊呼
- 回归真实语气；原本就不真实，建议不改写，而是建议用户别发
- **万能赞美词降密度** → 短段 <300 字里 `宝藏 / 神器 / 天花板 / 绝了 / 绝绝 / 绝绝子 / 好吃到想原地开店 / 气场两米八` 累计 ≥3 处时,只保留**紧扣具体事实**的那处(配合后面跟具体菜名 / 成分 / 价格的),其余改为具体描述或删掉
- **身份标签 + 反向 CTA 降密度** → `姐妹们快冲 / 不好吃回来打我 / 打工人必看 / 租房党必看 / 闭眼入` 组合 ≥2 处时,全篇最多保留一处;开头 + 结尾双号召的刚性模板必须拆掉一处
- **info-block emoji 微模板** → 保留信息槽位(`必点 / 地址 / 价格 / 晨晚护肤` 是合理的小红书阅读线索),但**删 emoji 前缀 + 把四字小标题压成自然短语**:`✅ 必点招牌` → `我点的几样`、`🌟 晨间护肤 / 🌙 晚间护肤` → `早上怎么洗 / 晚上怎么洗`、`✅ 技巧 1 / ✅ 技巧 2` → `技巧 1 / 2`(去掉 ✅)

**关于小红书 / 微博 hashtag 尾巴:**

真人小红书笔记 hashtag 平均 3-5 个、话题化(`#探店 #上海` 而非 `#小红书 X`)。AI 生成倾向堆 8-12 个 + 含**平台名冗余** `#小红书美食 / #小红书穿搭 / #时尚薯 / #吃货薯 / #美食薯 / #小红书爆款 X`。改写规则:

- **删:** 含 `#小红书 X / #X 薯 / #爆款 / #日常 / #分享` 的平台冗余标签 —— 平台名重复是 AI 信号(小红书系统不需要在内容里再标一次"这是小红书")
- **保留:** 话题化标签(`#上海美食 / #穿搭 / #护肤 / #租房改造`)
- **压到 3-5 个:** 按内容主次排序,只留最相关的

**例:** 原文 `#上海美食 #魔都探店 #面馆推荐 #平价美食 #吃货薯 #周末去哪儿 #我的美食日记 #小红书爆款美食`(8 个 + 2 个平台冗余) → 改后 `#上海美食 #探店 #面馆`(3 个,话题化)

> **语体限定：** 这条**只在自媒体 / 社交语体里激活**。商务、学术、公文、**品牌广告**里不会出现这些，不需要扫。**尤其注意:品牌广告 / 文案语体（Apple / Nike / 李宁 官网与 campaign）跟自媒体营销号不是同一个物种,不要把"#37 自媒体套路"往品牌文案上套**;详见"第零步 · 品牌广告 vs 自媒体营销号：三条识别线"。

> **关于"底层逻辑 / 颗粒度 / 心智"等词：** 这些词在**真·产品经理写 PRD、运营写复盘**时就是工作语言，不是 AI 腔。只有在**自媒体语境**里、且伴随"**伪深度结构**"（抽掉术语丢信息）时，才触发 #37-A。工作场景里不扫。

### 38. 文学 / 故事类 AI 味

**A. AI 故事套路开头：**
- `从前有座山,山里有座庙,庙里有个老和尚...`
- `在很久很久以前的一个小村庄里...`
- `那是一个风雨交加的夜晚...`
- `他永远忘不了那个下午...`

**B. AI 诗 / 对仗型文字：**
- 对仗工整但无感情：`月光如水水如天,思绪如云云如烟`
- 堆叠万能意象：`风、月、雨、雪、山、水、云、影`
- 主题空泛：`乡愁 / 岁月 / 时光 / 人生`

**问题：** 模板化开头 + 空意象 + 工整对仗 = AI 眼里的"文学感",但真文学靠**具体细节、观察、不规整的节奏**。

**改写方向：**
- 换掉模板开头，用具体时间、地点、人起句(`1978 年冬天,我爸在工厂门口抽烟`)
- 意象换成有指向性的观察（不是"月光",是"透过旧窗帘的那点光"）
- 允许不对仗、不整齐

> **语体限定：** 这条**只在文学 / 虚构 / 诗意类文本里激活**。非文学内容（论文、报告、邮件、朋友圈）不扫。

### 49. 网络流行语机械套用：毒性正能量缝合

> **核心辨识点：** 比起单纯的"密度 / 错位 / 过时",更常见、也更刺眼的是 **AI 把一个负面流行语缝到一个正能量结论上**,形成语气撕裂的"毒性正能量"。这是近年的高辨识度 AI 腔。

**核心问题：毒性正能量缝合**

真人用"打工人""内卷""躺平""班味"这些词,语气是**自嘲 / 反讽 / 疲惫 / 认命**。AI 生成小红书 / 公众号时,会把这些词当"接地气标签"拿过来,然后**强行接一个励志 / 反内卷 / 自我和解的结论**。读起来的感觉是:**词的情绪** 和 **句子的落点** 对不上。

**缝合句式(高辨识度 AI 腔):**

- `作为一个打工人/牛马,我想说:` + 积极行动建议
- `虽然班味很重,但只要 X,就能找回 Y` 型的励志转折
- `内卷是没有出路的,真正的答案是长期主义/自我价值/向内生长`
- `躺平不是终点,而是重新出发的起点`
- `吗喽的命也是命,所以我们更要 X`

真人讲累,讲完就算了,或者笑一下。真人不会"累完必升华"。每次必升华的,是 AI。

**三种诊断信号:**

1. **信号一 · 毒性正能量缝合** — 流行语 + 180° 转折 + 积极结论。见上方句式。**单次出现即警觉**,因为这是句式级的 AI 腔,不是密度级的。
2. **信号二 · 密度超标** — 一段 <200 字里流行语出现 ≥3 次,且集中在开头或每段首句。真人不会每三句一个 yyds。
3. **信号三 · 语境错位** — 商业计划书、政策段、学术文、PRD、述职报告里冒出 "yyds / 懂得都懂 / 吗喽",基本是 AI 强行"年轻化"的痕迹。

**关键词半衰期（2026-04 刷新，每年按《咬文嚼字》年度十大流行语 + 十大语文差错校准）**

| 分类 | 举例 | 现状 |
|---|---|---|
| **日常基础词（已沉淀，非 AI 腔）** | 破防、蚌埠住了（绷不住）、emo、打工人、内卷、躺平 | 真人日常用，不要仅凭出现就判 AI 腔；看的是**缝合**和**密度** |
| **《咬文嚼字》2024-2025 官方榜单** | **2024：**数智化、智能向善、未来产业、city 不 city、硬控、水灵灵地 xx、班味、松弛感、银发力量、小孩哥/小孩姐；**2025：**韧性、具身智能、苏超、赛博对账、数字游民、谷子、预制 xx、活人感、xx 基础 / xx 不基础 | 权威社会语言学观察，覆盖政策 / 生活 / 网络三层。真人 2024-2025 高频，AI 常滞后一两年才学会 |
| **社群常见鲜活词（榜单外）** | 抽象、牛马、吗喽、淡人 / 浓人、确诊为 xx、包 xx 的、已老实求放过、我的老天奶、沉默震耳欲聋 | 豆瓣 / 小红书 / 微博沉淀的口语表达，《咬文嚼字》未必收录，但真人高频使用 |
| **已入土（用了基本就是 AI 或训练数据老）** | yyds、绝绝子、栓Q、凡尔赛文学、人类高质量男性、家人们谁懂啊、姐妹们快冲、不好吃回来打我 | 真人现在基本不用；出现即高警觉。直播间 / 带货商单里用「家人们」是话术,不触发;AI 小红书 / 公众号正文用才触发 —— 鉴别看开头是否带 🆘 / emoji / 身份号召组合 |
| **梗 / 缩写** | 小丑竟是我自己、活久见、xswl、gkd、nbcs | 自然玩梗可保留，密集堆叠才是 AI 腔 |

**改写方向：**

- **毒性缝合** → 把升华那半句砍掉,留下流行语本身的情绪色调。累就累,不用接"但我还是相信自己值得更好"。
- **密度高** → 全段只留最合理的一处,其它改具体表达。"yyds" → 说清楚好在哪。
- **语境错位** → 正式文体里直接删,用符合语体的中性说法。
- **已入土词** → 换成 2025 鲜活词,或直接去掉,用朴素说法。

**原文(毒性正能量缝合 · 最典型的新 AI 腔):**
> 作为一个每天通勤两小时的打工人,我也曾班味浓得化不开。但后来我明白了:内卷是没有出路的,真正的答案在向内生长。与其焦虑,不如允许一切发生,慢慢把自己重养一遍。

**改后:**
> 我每天通勤两小时,回家躺床上手机都不想碰。不知道别人怎么样,反正我最近开始周三晚不加班,早回去遛狗半小时。没解决什么,但比我之前好点。

**原文(密度过高):**
> 这个功能真的 yyds！用完懂得都懂,姐妹们冲就完了！真的绝绝子,不冲我真的会谢。

**改后:**
> 这个功能挺顶的 —— 之前要点三层菜单,现在一键到底。

**原文(语境错位):**
> 本季度公司业绩实现重大突破,员工们表示 yyds,后续将继续保持这种状态。

**改后:**
> 本季度营收同比涨 23%,主要来自海外。

> **例外(不触发):**
> - 一段文字(>200 字)里自然出现**一次**流行语 —— 真人穿插,不是 AI 腔
> - 聊天 / 对话 / 朋友圈这种本就偏口语的文体,密度容忍度高
> - 自嘲型、反讽型使用且**不接升华** —— "今天班味真重,下班" 这种,真人写法,保留
> - 判断标准:这句话真人发出来,你会觉得合理吗?合理就保留。**但只要见到"负面词 + 升华转折"的缝合,几乎必是 AI 腔。**

### 50. B 站 AI 味(视频简介 / 科普解说 / 机械玩梗 / AI 字幕)

B 站是 2024-2025 AI 生成内容的重灾区之一。平台的内容生态(长视频、科普、解说、鬼畜、玩梗、弹幕文化)让 AI 生成有四种高辨识度痕迹。

**A. 视频简介 AI 模板(含时间轴幻觉)**

- 套路固定:**一句开场钩子 → 三行内容概要 → 时间轴 → 结尾求三连**
- **时间轴幻觉** —— AI 生成简介时常编出根本不存在的章节时间(`03:27 重点讨论 XX`),视频里那个时间点讲的完全是别的
- 结尾模板:`感谢大家的观看,如果这期视频对你有帮助,别忘了一键三连支持一下UP主哦!`

**B. 科普 / 解说稿(比喻 + 升华的 AI 体操)**

- **比喻成瘾:** "你可以把 X 理解成 Y" / "这就像..." —— 每段必来一个,而且比喻经常不贴切
- **强行上价值:** 讲一个具体技术 / 事件 / 人物,结尾必升华到"人类文明 / 科技浪潮 / 时代命题"
- **并列三件套结构:** "原因主要有三个:第一...,第二...,第三..." —— 真人口播会有犹豫、回头、补充;AI 稿平滑到反常
- **"接下来让我们一起来看看 / 下面我们就来聊一聊"** —— 口播过渡句的 AI 味重灾区

**C. 机械玩梗 / 电波错位**

- AI 硬套 B 站梗,但用错时代 / 用错语境:`AWSL`、`3Q`、`这集我看过`、`素质三连`、`爷青回` 用在不匹配的内容上
- **梗已入土还在用:** `真香` / `奥利给` / `哈基米` 套在 2025 的内容里,像两年前的复读
- **弹幕化台词混进正文:** 书面内容里突然来一句 `人类还在用爱发电吗` / `下次一定` —— 真人偶尔会,AI 会整段都是

**D. AI 翻译字幕残留**

- `我们...`、`我们的...` 主语代词密度异常(英文 `we/our` 直译,见模式 #33)
- 长定语句直译成一串"的": `一个被认为是...的...的...的 X`
- 半角标点混入全角环境(`,` 和 `，` 混用)
- 英文专有名词没译或乱译:`Transformer 模型(变压器模型)`、`context window(语境窗口)` 强行中译

**问题:** B 站受众对 AI 味敏感度高于小红书 / 抖音(观众平均年龄偏小、看视频时间长、会认真听旁白),一旦被识别,弹幕会直接打"AI 生成" / "味太冲"。

**改写方向：**

- **简介** → 删套模板,直接一两句说清楚这期讲什么;时间轴要么不写,要么对着视频真核对后再写(幻觉时间轴比没有时间轴糟)
- **口播稿** → 保留 UP 主的个人口头禅、停顿、跑题;把"强行升华"那段砍掉;比喻只留最贴切的一个
- **梗** → 只留跟具体内容真相关的一个;跨代际的梗先查近半年的 B 站热度再用
- **翻译字幕** → 按 #33 翻译腔残留办

**原文(AI 科普口播典型):**
> 大家好,今天我们来聊一聊 Transformer 模型。你可以把它理解成一个超级厉害的"翻译官"。它的核心机制主要有三个:第一是注意力机制,第二是位置编码,第三是前馈网络。可以说,Transformer 的出现,不仅改变了自然语言处理的格局,更是开启了人工智能的一个全新时代。感谢大家的观看,别忘了一键三连哦!

**改后:**
> 今天讲 Transformer。它最反直觉的地方是注意力机制 —— 以前的模型是挨个读词,Transformer 是**同时**看整句话,然后算每个词跟其它词的关系。位置编码是因为这样读没了先后顺序,得另外补一个。前馈网络那块我就不细讲了,不影响理解。2017 年这篇论文出来的时候,没人意识到它能把 NLP 掀成这样 —— 当时大家还在调 LSTM。

> **语体限定：**
> - UP 主真人风格(敬汉卿、老蒋巨靠谱、LKs 这类极强个人风格)**不要改**
> - 鬼畜区 / 整活区的节奏是故意反套路的,不触发 #50
> - 学习区 / 知识区 / 科普区的 AI 稿是 #50 主战场
