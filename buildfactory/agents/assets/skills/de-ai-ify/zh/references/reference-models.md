# 正面参考模型

> 本文件是 qu-ai-wei 的 **reference models**。用作**反向诊断**而非风格模仿 —— 让 LLM "按三联风格写"通常只会产出刻板仿作。SKILL.md 在自审阶段问"放进这些文本里会不会突兀",需要对照时读取本文件。
>
> **品牌广告 / 文案语体的深度参考与 5 组工作示例,见 [brand-voice.md](brand-voice.md),不在本文件。**

去掉 AI 腔只是一半；改写要落在读者熟悉的语域里。以下是几组**正面参考模型** — 但注意：**它们用作"对照诊断",不用作"风格模仿"**。

> ⚠️ 让 LLM "按三联风格写"之类的指令，通常只会产出刻板的仿作。本 skill 用这些名字来做**反向对照**（"这句话放进《三联生活周刊》会不会显得突兀？"），而不是作为模仿目标。

## 长读 · 长篇新闻与随笔（书面，有文学味，人的节奏）

- **《三联生活周刊》** — 现代中文书面语的基准：不僵硬、不花哨，具体、观察、像人。
- **《读者》** — 大众随笔/短文的基准，情感落地，抽象密度低。
- **《中国国家地理》** — 科学/自然/旅行长文，地名准确，不花哨 — 是"璀璨画卷"那种写景套话的反面教材。
- **《财新周刊》** — 财经/政策长文，克制、严谨。
- **《南方周末》** — 深度调查，句长节奏自然起伏。
- **《人物》杂志 / GQ 报道** — 叙事非虚构，场景细节密，抽象密度低。
- **《体坛周报》** — 体育长文 / 赛事评论的基准。遇到体育语境时用作对照（专有名词约定、惯用术语、叙事节奏都在这本里有参考）。

## 日常 · 产品/UX 白话（短句，接地气）

- **微信 UX 语言** — 所有简体中文用户每天都在看：通知、空状态、设置。短、中性、没有 emoji、没有"赋能/助力"、没有"让我们一起"。比如 `发送中...`、`网络未连接,请检查后重试`、`此消息已撤回`。
- **微信读书 UX** — 内容阅读类产品的同类语域。
- **小红书 笔记（非推广）** — 个人化、口语、第一人称，适合随笔语域。
- **豆瓣 长评** — 有想法但不装，个人声音强。

## 文学 · 文学散文（节奏与具体感）

扩展锚点集，覆盖平实/抒情/京味/南方/反思等语域。实际改写中按输入原文的语域挑最接近的文学对照。

- **汪曾祺 散文** — 短句、具体名词、没有修饰堆叠。`璀璨画卷`修辞的解毒剂。
- **阿城《棋王》《树王》《孩子王》** — 极简，几乎没有填充词，名词动词驱动。
- **沈从文《边城》《湘行散记》** — 抒情但具体，有地方感，不花哨。
- **老舍《骆驼祥子》《茶馆》** — 京味儿白话，幽默，有人物声口。
- **鲁迅 杂文 / 钱钟书《围城》** — ⚠️ **谨慎参考**。两人文字都有**有意的欧化倾向**（鲁迅五四早期白话实验，钱钟书大量西式长句与典故嵌套）。他们**不适合作为"去 AI 腔"的直接锚点** —— 强行模仿反而会学出"学者腔""文人腔",比 AI 腔更难祛除。只在**锋利度、密度、思辨强度**上借鉴，不在**日常白话自然度**上对照。
- **张爱玲《流言》《红玫瑰与白玫瑰》** — 细节具体，氛围感强。
- **余华《活着》《许三观卖血记》** — 平实、有力，没有修饰膨胀。
- **史铁生《我与地坛》《命若琴弦》** — 反思性散文，立足观察。
- **朱自清《背影》《荷塘月色》** — 经典，清晰，情感诚实。
- **林语堂 散文** — 机智，双语感，反掉书袋。
- **木心《文学回忆录》《素履之往》** — 格言式，压缩度高。
- **王小波 杂文** — 反讽，反装腔，聪明而好玩。
- **梁文道 / 唐诺 随笔** — 有文化密度，不堆辞藻。
- **贾平凹 散文 / 迟子建 散文** — 秦地 / 东北方言感，地域具体。

## 公文 / 学术 / 法律 语体锚点

去 AI 腔 **不等于** 把所有文本降级到日常白话。正式语体有自己必须的写法 —— 当文本属于这些语体时，**不应套用"日常自然"的标准**。

**公文 / 政论：**
- 《国务院公报》— 现行政府公文语体范本
- 《人民日报》社论 — 政论体权威
- 《最高人民法院公报》— 司法公文的精确度范本

**学术 / 科技：**
- 《中国社会科学》《经济研究》《中国科学》系列期刊 — 一流学术语体
- 商务印书馆《现代汉语八百词》（吕叔湘）、《语法讲义》（朱德熙）— 语法学权威著作本身就是学术语体范本

**法律文本：**
- 《中华人民共和国民法典》《刑法》— 法律语言的精确度标准
- 《最高人民法院指导性案例》— 司法文书范本

**怎么用：** 同样作"反向诊断"而非"风格模仿"。改写前问："如果这段文字出现在这类正式文本里，会突兀吗？"会突兀 → 这个改写方向错了，退回来。

## 怎么用这些参考

1. **在第 3 步（自审）中：** 对每一处被标记的句子问一遍 — "这句话放进《三联生活周刊》或《微信》系统提示里，会显得突兀吗？"会，就确认是 AI 腔。**用作反向参照**（不会出现在这些文本里），**不作为正向模仿**。
2. **v1 默认改写语域：日常中性白话** — 现代普通话，不用装饰词，不用宣传腔，不用文艺腔。风格模仿（"写得像三联","写得像汪曾祺"）**推迟到 v1.1**,到时候会放入真实短例作为对照，而不是靠命名。
3. **语音校准回退：**用户没提供样本时，默认 `日常中性白话`。v1 **不**提供"三联风 / 微信UX风 / 汪曾祺风"这类一键风格。
