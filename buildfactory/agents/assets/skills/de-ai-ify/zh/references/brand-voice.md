# 品牌广告 / 文案语体（深度参考）

## 为什么品牌广告 / 文案要单独列一种语体

好的品牌本地化文案（Apple CN 官网、Nike 大中华区 campaign、NB、李宁品牌大片、小米发布会大字）跟这个 skill 里其他 8 种语体**全都不是一个物种**:

- **不是自媒体营销号** —— 没有 KOL 人设、没有"姐妹们冲"、没有点赞收藏 CTA、没有伪深度咨询腔；品牌是企业级单向输出,不是"我"在跟你聊。
- **不是商务 / 职场** —— 品牌文案避免"赋能 / 助力 / 打造 / 闭环"这类商务僵尸词,它们恰好是品牌文案要**反着写**的东西。
- **不是书面 / 一般** —— 品牌文案句子极短（常 2-6 字）、允许不完整句、允许留白做节奏、允许文言感（`岂止于大` / `让妈,更妈`）、允许中英混排（`iPhone 跟 Mac` 直接留英文名）,这些放进博客时评里全都不合规。

### 品牌文案的语体特征

1. **短到不能再短**:`Just Do It`(Nike)、`岂止于大`(Apple iPhone 6 Plus)、`让妈,更妈`(Apple 母亲节)、`管什么输赢,上`(Nike)
2. **祈使动词 + 物理动作**:`上`、`冲`、`跑`、`跳`、`敢` —— 不是`激发 / 赋能 / 助力`
3. **允许(甚至鼓励)排比、韵律、文言感** —— 这些在书面语里是 AI 腔信号,在品牌语境里是**刻意设计的修辞**,要保留
4. **允许留白与不完整句** —— `轻。薄。快。`、`没状态?上。没把握?上。` —— 这些在#35 句长均质化里是"短句过短",但品牌语体里是节奏
5. **中英可混用,且英文专名**一律保留** —— `Apple Intelligence 登场`、`为 iPhone 打造`,不翻译成"苹果智能"
6. **文案里没有"我"(KOL 人设),只有"你"(品牌对你说)或无主语祈使**

**所以品牌文案要单列一档**,否则按其他 8 种语体任何一种都会误改：当商务会被加回"赋能助力";当自媒体会被塞回"家人们";当书面会被改成完整长句,破坏节奏；当特稿会被扩展成段落,杀掉留白。**判断不清时,问用户:"这是要发在品牌官网 / 广告片 / 发布会大字 / 户外大牌上吗?"** 是 → 切品牌广告语体。

## 品牌广告 vs 自媒体营销号：三条识别线

这两个语体最容易混——都偏消费端、都"卖东西"。但它们是不同物种,激活的规则子集几乎反向。区分看三条：

| 信号 | 品牌广告 / 文案 | 自媒体营销号 (#37) |
|---|---|---|
| **说话的人** | 企业级单向输出（Apple / Nike 作为 brand voice 对消费者说），**没有"我"作为 KOL 存在** | 有强人设（"25 岁裸辞第 30 天的我" / "作为一个打工人"），**第一人称 KOL 视角** |
| **CTA 形式** | `Just Do It` / `上` / `敢` / `为更好而生` —— **祈使 + 行动动词** | `点赞收藏关注` / `评论区告诉我` / `建议收藏` / `一键三连` —— **流量导向** |
| **信息密度** | **极度压缩**（常 2-6 字一句）,留白是美学 | **填充式**（痛点 → 方法 → 升华）,文字堆得很满 |

**四个边界场景：**

- 品牌**自己运营的公众号推送**,如果带 KOL 语气("今天想跟大家聊聊...")→ 走 **自媒体语体**,不是品牌广告
- 品牌**PR 新闻稿 / 年报 / 公关声明** → 走 **商务 / 书面语体**,不是品牌广告
- 品牌 **SCRM 社群 "亲~" 话术** → 走 #37-E 的 "AI 客服亲亲体",不是品牌广告
- 品牌 **官网 hero copy / 发布会大字 / 广告片旁白 / 户外大牌** → **品牌广告语体**

## 品牌广告语体下每条规则的特殊判定

品牌广告 / 文案的雷区最多，几乎每一类都要重新判（这正是它必须单独列一档的原因）：

- **A 内容拔高**:`#1` 过度拔高意义(`标志着...里程碑 / 谱写新篇章`) 和 `#4` 尾巴式空洞总结要删；但 `#5 模糊权威归因` 在品牌文案里**不触发**（品牌自己就是权威,不需要"业内人士"背书）；`#6 挑战与展望` 一般品牌文案里不出现,遇到基本是 AI 仿品牌稿。
- **B 语言**:`#7 AI 高频词`（赋能 / 助力 / 打造 / 闭环 / 护航）**严查、全删** —— 这些词是 AI 模仿品牌文案的第一破绽,真 Apple / Nike 文案从不用；`#11 的的不休`、`#18 过度数字化列点`、`#19 空枢纽`、`#20 列表预告` **不触发**（品牌文案本来就短、不列点）；`#17 中英术语强行括注` **不触发**（品牌文案 `iPhone`、`Mac`、`Air Max` 直接留英文,这是正确做法）。
- **C 修辞**:`#21 四字成语堆叠`、`#22 排比段` 要**格外克制地判**。排比是品牌文案最核心的修辞（`管什么输赢,上` / `跑了就懂` / `没状态?上。没把握?上。他们说你不行?上。`）,刻意设计的韵律**不是 AI 腔**。判断标准同特稿:**紧扣情绪和行动 vs 脱离内容空转**。`#24 过度加粗与 emoji` **触发**（品牌文案不用 emoji）；`#25 破折号滥用` 要看场景（Apple 文案里 `——` 常见且克制）。
- **E 填充**:`#32 "让我们一起..." 号召句` 要**小心区分**。`Just Do It` / `上` / `敢敢的` 是**品牌 CTA**,不是 AI 空号召；AI 仿品牌写的 `让我们一起开启精彩新篇章` 才是 #32 要改的。判据:号召语是否**短、具体、带行动动词**。
- **G 篇章**:`#35 句长节奏均质化` **反向触发** —— 品牌文案要的恰恰是节奏变化、允许超短句和留白；`#36 指代不敢省` 在品牌文案里**不触发**（品牌名有时必须反复出现做记忆强化,`iPhone × 3` 是常规操作）。
- **H 平台场景**:`#37 自媒体套路` **不触发**（品牌文案跟 KOL 自媒体不是一个物种,见上文"品牌广告 vs 自媒体营销号：三条识别线"）；`#38 文学 AI 味` **触发** —— 堆意象（`璀璨 / 乐章 / 华章 / 风月雨雪`）是 AI 仿品牌广告的常见破绽。

**一句话总结品牌文案规则**:**严查翻译腔和 AI 高频商务词,严防堆意象,但允许并保护短句、留白、排比、文言感、中英混排。**

## 品牌广告 / 文案语体锚点

品牌广告的中文本地化做得最好的两家是 **Apple 大中华区**和 **Nike 大中华区**。它们是诊断"这段文字是不是 AI 写的品牌文案"的**反面参照基准**——如果改写后的句子放进 Apple 官网 hero copy 或 Nike 大中华区 campaign 里会刺眼,这个改写方向就错了。

### Apple CN 官网文案

- 经典例：`岂止于大`（iPhone 6 Plus，"Bigger than bigger" 的中文版，文言感 + 留白,不是直译"比大更大"）
- 经典例：`让妈,更妈`（2024 母亲节 campaign，`Make mom mom-er` 式的创造性结构）
- 经典例：`唯一的改变,是一切都变了`（iPhone 6s，"The only thing that's changed is everything"）
- 经典例：`超越前所未见。非同凡响。`（iPad，极短句 + 不完整句 + 句号收尾）
- 特征：极短、允许文言感、产品名一律留英文（`iPhone` / `Mac` / `AirPods`）、标点用句号不用感叹号、没有 emoji、没有"赋能 / 助力 / 打造"

### Nike CN campaign 文案

- 经典例：`管什么输赢,上。`（2020 疫情后回归 campaign）
- 经典例：`跑了就懂。`（跑步 campaign）
- 经典例：`活出你的伟大。`（`Find Your Greatness`）
- 经典例：`没状态?上。没把握?上。他们说你不行?上。输了?明天接着上。`（排比,但每句紧扣具体情境,不是空排比）
- 经典例：`Just Do It`（英文原版直接保留,不翻成"就去做吧"）
- 特征：祈使动词 + 物理动作（上 / 冲 / 跑 / 跳）、反复用疑问 → 单字回答的节奏、句号而非感叹号、**允许直接留英文 slogan**、不用"您"

### 其他可参考品牌

- **小米发布会大字**（雷军式发布会 slide 文案,短、具体、有数字）
- **李宁**（国潮品牌的中文自创 slogan）
- **NB 大中华区**（慢跑、经典复古语境）
- **支付宝 / 淘宝 UX 文案**（消费级产品的品牌级微文案）

### 怎么用

1. **反向诊断**:改写后的句子放进 Apple 官网或 Nike 广告片里,会不会显得假?会 → 改写方向错,回退。
2. **四个硬信号检查**——真·品牌文案都符合,AI 仿品牌文案至少破一条:
   - 短到不能再短（每句 2-8 字为主,超过 12 字要警觉）
   - 没有 `赋能 / 助力 / 打造 / 一体化 / 全方位 / 多维度`
   - 产品名保留英文（`iPhone` 不是"苹果手机","Mac" 不是"苹果电脑"）
   - 标点克制（句号为主,感叹号慎用,emoji 绝不用）
3. **不做正向模仿**:让模型"按 Apple 风格写"或"写个 Nike 式 slogan"通常只会产出刻板仿作。真·品牌文案背后是品牌战略、用户洞察、拍摄制作,不是一句话生成。

## 5 组工作示例（Apple / Nike 风）

下面 5 组「原文 / 改后」对照演示**品牌广告语体**的改写方向。**重点：不要把这个语体里的"改后"当成默认改写模板往其他语体上套**——它们只在品牌官网 hero copy / campaign 主视觉 / 发布会大字 / 户外大牌 / TVC 旁白这些场景才成立。

> ⚠ **输出格式说明:** 本节 5 组示例演示**品牌语体的语言 register**(短句 / 留白 / 文言感 / 中英混排),不演示完整输出流程(初稿 → 自审 → 终稿 → 打磨报告 → 改动摘要)。这里为了聚焦语体差异,只保留改动摘要;实际使用时,品牌语体的终稿同样要附「打磨报告」,格式见 SKILL.md「主动打磨 · 打磨报告」章节。

### 示例 1：产品发布 hero copy（AI 仿 Apple 最常见的翻车）

**原文（AI 腔品牌文案）：**
> 全新 XX 手机,搭载了最新一代的先进处理器,为您带来前所未有的极致性能体验。无论是工作还是娱乐,都能让您得心应手,尽在掌握。重新定义您对移动设备的想象,开启智慧生活新篇章。

**改后（品牌广告语体）：**
> 新一代芯片。
> 更快。更省电。
> 日常应用,快得你会忘了在用它。

**改动摘要（品牌文案视角）：**
- 删 `搭载了最新一代的先进处理器` → 直接 `新一代芯片`（品牌级文案不用"搭载"这种 TO B 公关腔）
- 删 `前所未有的极致性能体验 / 得心应手 / 尽在掌握 / 重新定义 / 开启智慧生活新篇章` 全部 —— AI 高频词 + 尾巴式空洞总结
- 删 `您` → 品牌广告用 `你`,距离更近（Apple CN 文案固定用"你",不用"您"）
- 长句拆短,留白做节奏
- 保留具体体感（`快得你会忘了在用它`）—— 具体 > 抽象

### 示例 2：campaign 主视觉文案（Nike 风运动)

**原文（AI 腔运动文案）：**
> 我们坚信,通过不懈的努力和拼搏,每一位运动员都将实现自我超越,谱写属于自己的辉煌篇章。让我们一起,见证这场关于梦想与荣耀的伟大征程!

**改后（品牌广告语体）：**
> 没状态?上。
> 没准备好?上。
> 他们说你不行?上。
> 输了?明天接着上。

**改动摘要：**
- 删 `我们坚信 / 通过不懈的努力 / 实现自我超越 / 谱写属于自己的辉煌篇章 / 让我们一起 / 见证这场关于梦想与荣耀的伟大征程` —— 全都是 AI 生成体育励志文案的标配成分
- 改为疑问 → 单字祈使的节奏（品牌广告常见结构,每句紧扣具体情境,不是空排比 —— 通过了#22 排比"紧扣具体场景"的判据）
- 允许四个短句连排 —— 在品牌广告语体里是节奏设计,不是 #35 均质化

### 示例 3：母亲节 / 节日 campaign（Apple 创造性结构）

**原文（AI 腔节日文案）：**
> 在这个充满爱的特别日子里,让我们一起回忆妈妈为我们付出的点点滴滴,感恩她给予我们的无私的爱与温暖。愿每一位妈妈都能收获属于她的幸福与美好!

**改后（品牌广告语体）：**
> 她是妈。
> 也不只是妈。
> 今天,把镜头对准她。

**改动摘要：**
- 删 `在这个充满爱的特别日子里 / 让我们一起回忆 / 感恩 / 无私的爱与温暖 / 愿每一位妈妈都能收获属于她的幸福与美好` —— 节日文案 AI 腔模板
- 用创造性结构（`她是妈 / 也不只是妈`）替代空抒情 —— 这种短句嵌套回绕是 Apple CN "让妈,更妈" / "岂止于大" 那一类文案的招牌结构
- 收尾给具体行动（`把镜头对准她`）—— 产品是相机/手机,所以 CTA 落到一个具体动作

### 示例 4：户外大牌短 slogan

**原文（AI 腔 slogan）：**
> 卓越品质,智慧生活,助力您开启未来科技之旅!

**改后（品牌广告语体）：**
> 为更好而生。

或

> 你要的未来。已经在你手里。

**改动摘要：**
- 删全部 AI 高频商务词（`卓越品质 / 智慧生活 / 助力 / 开启未来科技之旅`）
- 极度压缩。真·户外大牌文案通常 ≤ 10 字
- 允许不完整句（`你要的未来。已经在你手里。`）—— 句号做节奏停顿

### 示例 5：产品介绍（处理中英混排）

**原文（AI 腔产品介绍,强行中译）：**
> 搭载变压器模型架构的智能芯片,配合全新空间计算视窗,让您体验前所未有的智慧交互。

**改后（品牌广告语体）：**
> Transformer 架构。全新空间计算。
> 交互,重新想象。

**改动摘要：**
- `变压器模型` → `Transformer` —— 品牌文案中**专业术语保留英文**（`Transformer` / `iPhone` / `Apple Intelligence` / `Retina` 绝不翻译,这是品牌本地化硬规则,不是 #17 中英括注）
- `搭载 / 配合 / 智慧交互 / 让您体验前所未有的` —— AI 高频词 + 客服腔全删
- 拆成短句 + 句号收尾

> ⚠️ **这 5 个示例的"改后"版本,只在品牌广告语体里适用。** 把这种极压缩短句风格套到博客、特稿、学术论文、工作邮件上,会违反那些语体的规范(工作邮件里写"新一代芯片。更快。更省电。"就像在会议上只说关键词、没有完整汇报)。**语体识别先行,改写风格后选。**
