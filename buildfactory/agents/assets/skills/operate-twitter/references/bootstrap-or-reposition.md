# Bootstrap or reposition the account

Use this playbook when the live profile is blank, incomplete, internally
inconsistent, or attached to a company direction it no longer represents.

## Establish the account promise

Read the company evidence before writing a bio. Recover the actual product or
work, the concrete people it serves, the current objective, useful customer
language, and any identity the company already uses elsewhere. Do not invent a
generic founder persona or force every company into a personal-brand format.

Decide what a visitor should understand from the profile without scrolling:

- who is speaking;
- what the company is building or learning;
- who should care;
- what credible next step or destination exists now.

If those answers are not supported by company evidence, investigate or leave
the uncertain field alone. A polished unsupported identity is worse than an
honest incomplete one.

## Audit the live first screen

Inspect the current display name, handle, bio, avatar, banner, location, URL,
pinned post, and the first visible posts. Judge them as one surface: each item
may be individually plausible while the combined profile tells no coherent
story.

Choose the smallest set of changes that makes the surface coherent. An account
does not need every optional field, and account setup is not a fixed checklist
that must manufacture work.

## Produce missing material

- Use `mine-customer-voice` when the audience language or positioning is still
  conjecture.
- Use `design-asset` or `gen-image` for an avatar or banner only when a visual
  would carry real identity information.
- Run `visual-iterate` before publishing generated visual material.
- Run `de-ai-ify` on public-facing bio or pinned copy.

Keep account-specific facts in company state, never in this skill.

## Apply and verify

Capture the live starting values before editing. Reconfirm the current handle,
make one coherent group of changes, then reopen the public profile and judge
the rendered result at normal desktop width. Verify the actual link target,
image crop, bio truncation, handle, and pinned content rather than trusting the
edit form.

A handle change affects existing profile links and references. If it is truly
needed, update durable company facts that still point at the old handle after
the live change is confirmed.

If the profile needs a first body of content, use the publishing playbook. Do
not set a permanent bootstrap mode or require an arbitrary number of seed
posts; the next Goal will reassess the live account again.
